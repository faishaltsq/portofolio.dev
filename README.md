# portofolio.dev

Personal portfolio website for Muhammad Faishal Tsaqief.

Focused on product quality, test automation, API validation, and engineering collaboration.

## Live

- GitHub Pages: `http://faishaltsq.github.io/portofolio.dev/`

## About

This portfolio presents:

- Quality engineering background
- Manual, exploratory, regression, and API testing
- Playwright automation workflow
- Selected projects and technical stack

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Features

- Retro-inspired UI with modern layout
- Responsive desktop and mobile design
- Structured sections for profile, testing, automation, experience, projects, skills, and contact
- Hover animations for cards, buttons, and links
- Static export support for GitHub Pages

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is configured for GitHub Pages static export.

Workflow:

- `.github/workflows/deploy-pages.yml`
